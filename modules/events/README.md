# Events Module

Module de gestion des événements spéciaux Pokémon GO.

> 📖 **Documentation complète** : Voir `docs/events/` à la racine du plugin

## Fichiers de Documentation

Toute la documentation se trouve dans `docs/events/` :

- **`README-ROUTING.md`** - Documentation du système de routing
- **`INTEGRATION-ELEMENTOR.md`** - Intégration avec Elementor
- **`EVENEMENTS-DISTANTS.md`** - Sources d’événements (local / distant / SQL), routing, Spotlight Day Hours

## Structure du Module

- `events.php` - Fichier principal
- `admin/` - Interface d'administration
- `functions/` - Fonctions helper et queries
- `public/` - Routing front-end et shortcodes
